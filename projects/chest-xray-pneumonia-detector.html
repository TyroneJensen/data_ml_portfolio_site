Project Overview

This project implements a deep learning model to detect pneumonia from chest X-ray images. It uses transfer learning with a pre-trained ResNet18 model and provides a user-friendly web interface for real-time predictions.

Features

  • Upload and analyze chest X-ray images
  • Real-time pneumonia detection
  • Probability scores for normal and pneumonia cases
  • Interactive web interface
  • Pre-trained ResNet18 model fine-tuned on medical images

Technology Stack

  • Python 3.11
  • PyTorch (Deep Learning)
  • Torchvision (Image Processing)
  • Gradio (Web Interface)
  • ResNet18 (Pre-trained Model)

Model Architecture

  • Base: ResNet18 (pre-trained on ImageNet)
  • Modified for binary classification
  • Input size: 224x224 RGB images
  • Output: Binary classification (Normal/Pneumonia)

Dataset

  • 5,863 X-Ray images (JPEG)
  • 2 categories: Normal and Pneumonia
  • Training and testing sets

Performance

  • Validation Accuracy: ~90%
  • Real-time inference
  • Balanced precision and recall
