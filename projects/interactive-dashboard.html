Project Overview

A comprehensive data visualization platform that transforms complex medical data into statistical insights. The dashboard leverages modern web technologies to provide real-time analytics, interactive reporting, and intuitive data exploration.

Features

  • Real-time data visualization
  • Interactive filtering and drill-down capabilities
  • Multi-dimensional data analysis
  • Responsive design for desktop and mobile

Technology Stack

  • Python
  • Tableau
  • Google Earth

Data Processing

  • ETL pipelines with CSV and Python
  • Data cleaning and transformation using Pandas
  • Data augmentation using the Google Earth API
  • EDA visualizations using Seaborn and Matplotlib
  • Automated data refresh mechanisms

Performance

  • Low-latency data rendering
  • Supports large-scale datasets

Dataset

  • Uses custom dataset from Western Cape Blood Service
  • Features include: Age, Sex, Race, Ferritin, Hemoglobin
  • Geospatial data obtained from custom compiled database of co-ordinates using Google Earth
  • Multiple clinical parameters for analysis and prediction
