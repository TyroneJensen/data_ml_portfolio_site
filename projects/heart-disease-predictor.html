Project Overview

This project implements a machine learning web application that predicts the probability of heart disease based on patient data. Built with scikit-learn and Gradio, it provides an intuitive interface for real-time predictions.

Features

  • Interactive web interface for heart disease prediction
  • Machine learning model trained on UCI Heart Disease dataset
  • Real-time predictions with probability scores
  • Simple and intuitive user interface

Technology Stack

  • Python 3.11
  • scikit-learn (Machine Learning)
  • Gradio (Web Interface)
  • Pandas (Data Processing)
  • NumPy (Numerical Operations)

Model Performance

  • Algorithm: Logistic Regression
  • Accuracy: 87% on test set
  • Features: 13 clinical parameters

Dataset

  • Uses the UCI Heart Disease dataset
  • Features include: Age, Sex, Chest Pain Type
  • Blood Pressure and Cholesterol Levels
  • Multiple clinical parameters for prediction
